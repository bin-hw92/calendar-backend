# calendar-backend

### 할 일 공유 달력 서버(백엔드 기술)
- Koa 사용
- MongoDB 사용

### 프론트단 기술(calendar-frontend 프로젝트 내용)
  - React 사용
  - Redux-saga 사용
  - css 사용 반응형 웹으로 개발


### 현재 기능 및 앞으로 구현 내용
  - [x] 달력 화면 구현, 반응형으로
  - [x] 해당 날짜에 할 일 등록
  - [x] 여러 날짜에 걸쳐 등록 시 화면에 줄로 표시
  - [x] 라벨 기능 추가
  - [ ] 오늘 버튼 추가
  - [ ] 할 일 공유용 게시판 추가
  - [ ] 게시판 입장 시 달력화면으로 이동 
  - [ ] 게시판 비밀번호 기능
